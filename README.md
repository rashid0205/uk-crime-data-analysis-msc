# UK Crime Data Analysis (MSc Master Project)

## Overview
This repository contains my MSc Master Project focused on analysing UK Metropolitan Police street crime data from 2022 to 2025.  
The project explores spatial and temporal crime patterns across London using data analytics and visualisation techniques.

## Dataset
- Source: UK Metropolitan Police (data.police.uk)
- Time period: 2022â€“2025
- Coverage: London boroughs and LSOA-level street crime data
- Note: Only sample or processed data is included in this repository.

## Objectives
- Identify crime trends over time
- Analyse crime distribution by borough and LSOA
- Explore seasonal and time-based crime patterns
- Produce clear visual insights for analytical decision-making

## Methodology
- Data cleaning and preprocessing using Python (Pandas, NumPy)
- Exploratory Data Analysis (EDA)
- Feature engineering (time, location, crime type)
- Data visualisation using Matplotlib and Seaborn
